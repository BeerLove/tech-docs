User and Group Managment
++++++++++++++++++++++++

Key Goals
=========
* Managing users' and groups' (local) life cycle
* Manage passwords
* Identify linux users

Content
=======
* Identify linux users:
    * id

* Managing users:
    * /etc/passwd
    * useradd
    * usermod
    * userdel
    * chsh

* Managing users' passwords:
    * /etc/shadow
    * passwd
    * chpasswd
    * chage

* Users's defaults:
    * /etc/login.defs

* Managing groups:
    * /etc/group
    * groupadd
    * groupmod
    * groupdel
    * newgrp
    * chgrp
 
* Managing groups' passwords:
    * /etc/gshadow
    * gpasswd
